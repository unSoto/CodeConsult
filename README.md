## ⚡ QUICK START

```bash
# 1. Clone the project
git clone [REPOSITORY-URL]

# 2. Start local server
cd code-consulting
python -m http.server 8000

# 3. Open in browser
# http://localhost:8000
```

**ALTERNATIVE START METHODS:**
```bash
# Via Node.js
npx http-server

# Via PHP
php -S localhost:8000
```
