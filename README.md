## âš¡ Quick Start

### Requirements
- Web browser with HTML5 support
- Local web server (optional)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/unSoto/CodeConsult.git
   cd CodeConsult
   ```

2. **Start the server:**
   ```bash
   # Option 1: Python 3
   python -m http.server 8000

   # Option 2: Node.js
   npx http-server -p 8000

   # Option 3: PHP
   php -S localhost:8000
   ```

3. **Open in browser:**
   ```
   http://localhost:8000
   ```
